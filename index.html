Elementos en bloque

Un elemento de bloque HTML es aquel que ocupa todo el ancho de la página o del elemento que lo contiene. Los elementos de nivel de bloque ocupan tanto espacio como sea posible de forma predeterminada . Cada elemento de nivel de bloque iniciará una nueva línea en la página y se apilará hacia abajo. Además de apilarse verticalmente, los elementos de nivel de bloque también ocuparán tanto espacio horizontal como sea posible. El elemento p es un ejemplo de un elemento de nivel de bloque.

Elementos en línea

Un elemento en línea en HTML o inline elements en inglés, es una etiqueta que se coloca dentro del flujo de texto y solo ocupa el espacio que necesita para mostrar su contenido. A diferencia el elemento en bloque, no ocupa todo el ancho disponible de su contenedor padre.
